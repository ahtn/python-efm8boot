efm8 boot
===============================

version number: 0.0.1
author: jem

Overview
--------

A python package for working with efm8 bootloaders.

Installation / Usage
--------------------

To install use pip:

    $ pip install efm8boot


Or clone the repo:

    $ git clone https://github.com/ahtn/efm8boot.git
    $ python setup.py install
